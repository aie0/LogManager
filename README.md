NodeJS log manager
==============

NodeJS log manager with session tracking integrated with with [Winston][0] and [Bunyan][1]
--------------

**Main Features:**
- Request id persistence
- Operation delta

For more info, please read Deep Dive JavaScript article - [Logging in Node.js][2]
